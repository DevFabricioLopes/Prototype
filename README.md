# Prototype

Implementação do padrão de projeto Prototype em Java.

## Descrição

O padrão Prototype permite copiar objetos existentes sem depender de suas classes concretas, facilitando a clonagem de objetos complexos.

## Estrutura

- Interface Prototype
- Implementações concretas com método de clonagem

## Como usar

Clone o repositório e execute as classes de exemplo para ver a clonagem em ação.
